# NETC-240 Emerging Topics in Computer Network Engineering Technology
## Course Description
A course on current industry needs related to Computer Network Engineering Technology. Topics include: AI, MS 365 Administration, cloud computing, and Linux.

## Assignments Overview
Labs are thorough explorations of the weekly topics. Most include screenshots and analysis questions.

## Weekly Syllabus
- 	Week 1 – Cloud Computing - Azure and AWS
- 	Week 2 – Cloud Computing - Secure VM Configuration
- 	Week 3 – Cloud Computing - BLOB Storage / Linux - Basic Commands
- 	Week 4 – Linux - CLI and Bash Scripting
- 	Week 5 – Linux - Security / Intro to AI
- 	Week 6 – AI - Prompt Engineering and LLM Models
*- 	Week 7 – Midterm*
- 	Week 8 – MS 365 - File Sharing and Teams
-	Week 9 – MS 365 - Teams, OneDrive, and MS 365 Licenses
- 	Week 10 – MS 365 - Forms and Power Automate
- 	Week 11 – MS 365 - SharePoint and Outlook
- 	Week 12 – MS 365 - Purview and eDiscovery
- 	Week 13 – MS 365 - Intune
- 	Week 14 – MS 365 - Entra and Security
*- 	Week 15 – Final*
