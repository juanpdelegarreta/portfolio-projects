# NETC-170
## Course Description
A course on frameworks for organizational governance of information technology. Topics include: IT portfolio management, risk and compliance, and business continuity planning and impact analysis.

## Assignments Overview
Module assignments include research, documentation, and applying concepts to create governance-type documents.

## Modulely Syllabus
- 	Module 1 – What Makes a Great Leader?
- 	Module 2 – Mission vs Vision Statements
- 	Module 3 – IT Strategy Alignment
- 	Module 4 – Security Controls
- 	Module 5 – Cybersecurity Compliance
- 	Module 6 – Privacy Compliance
- 	Module 7 – Security Controls
- 	Module 8 – Metrics
-	Module 9 – Primary Goals
- 	Module 10 – Business Impact Analysis
