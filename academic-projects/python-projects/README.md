# IT-115 - IT Scripting
## Course Description
A course on task automation and configuration management using Python programming language. Topics include: modifying existing Python scripts, and creating new scripts to automate common tasks.

## Assignments Overview
Lab Assignments include questions and specific Python scripts/programs

## Weekly Syllabus
- Week 1	Intro to Programmning and Python
- Week 2	Simple Data Types, Python Basics
- Week 3	Complex Data Types - Lists, Tuples, Sets
- Week 4	Complex Data Types - Dictionaries, Json, Yaml
-  Week 5	Flow Control (if, while, for)
-  Week 6	Functions
-  Week 7	Extending Python with packages
-  Week 8	Virtual Environments, User input, Mid-term
-  Week 10	File Handling, CLI, GUI
-  Week 11	Pattern Matching/Regex, Error Handling, Testing
-  Week 12	APIs
-  Week 13	Data Visualization
-  Week 14	Network Automation
-  Week 15	Network Automation
